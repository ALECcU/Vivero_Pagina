# 🌿 Vivero El Granjero — Landing Page Premium

> **"El paraíso al alcance de tus manos"**

Landing page corporativa de alto impacto para **Vivero El Granjero**, desarrollada con HTML5, CSS3 y JavaScript vanilla. Diseño Dark Luxury adaptado al sector de viverismo y jardinería.

---

## 🚀 Cómo abrir la web

1. Asegúrate de tener la siguiente estructura de archivos:
```
Pagina_web/
├── index.html
├── styles.css
├── script.js
├── README.md
└── images/
    ├── logo.png         ← TU LOGO (cópialo aquí si no está)
    ├── hero_visual.png
    ├── service_1.png    (Plantas ornamentales y tropicales)
    ├── service_2.png    (Sustratos especializados)
    ├── service_3.png    (Macetas y contenedores)
    ├── service_4.png    (Asesoría personalizada)
    └── service_5.png    (Distribución a viveros)
```

2. Abre `index.html` en tu navegador web (Chrome, Edge, Firefox)
3. Para verla en red local, puedes usar VS Code con la extensión **Live Server**

## 🔧 Personalización

### Cambiar información de contacto
En `index.html` busca:
- WhatsApp: `https://wa.me/573000000000` → reemplaza con tu número
- Email: `contacto@viveroelgranjero.com` → tu email real
- Dirección: "Tu ciudad, Colombia" → tu dirección real
- Horario: ajusta según tu establecimiento

### Cambiar textos
Todos los textos están directamente en `index.html`, bien comentados y organizados por sección.

### Cambiar colores
En `styles.css`, modifica las variables CSS en `:root {}` al inicio del archivo.
